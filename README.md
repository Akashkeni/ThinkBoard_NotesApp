# 📝 NotesApp - ThinkBoard

A full-stack Notes Application built using the MERN stack (MongoDB, Express, React, Node.js) with modern production-level features like rate limiting, REST API architecture, and responsive UI.

---

## 🚀 Features

- 🧠 Create, Read, Update, Delete (CRUD) Notes
- ⚡ RESTful API with Express.js
- 🗄️ MongoDB database with Mongoose
- 🔐 Rate Limiting using Upstash Redis
- 🌐 Fully responsive React frontend
- 🧩 Clean MVC backend architecture
- 📦 Environment-based configuration (dotenv)
- 🚀 Ready for deployment
